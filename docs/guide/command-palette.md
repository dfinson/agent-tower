# Command Palette

The command palette provides quick navigation and search across all jobs.

## Opening the Palette

Press `⌘K` (Mac) or `Ctrl+K` (Windows/Linux) from anywhere in the UI.

## Features

- **Search jobs** — Type to filter by job ID, title, repository, or branch
- **Quick navigation** — Select a result to jump directly to that job
- **Keyboard-driven** — Arrow keys to navigate, Enter to select, Escape to close

## All Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Alt+N` | Create new job |
| `⌘K` / `Ctrl+K` | Open command palette |
| `⌘,` / `Ctrl+,` | Open settings |
| `` Ctrl+` `` | Toggle terminal drawer |
| `Ctrl+Enter` | Submit prompt or message |
| `/` | Focus search / filter jobs (on dashboard) |

See [Keyboard Shortcuts Reference](../reference/keyboard-shortcuts.md) for the complete list.

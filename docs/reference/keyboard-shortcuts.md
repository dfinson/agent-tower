# Keyboard Shortcuts

## Global Shortcuts

These work from anywhere in the CodePlane UI.

| Shortcut | Action |
|----------|--------|
| `Alt+N` | Create new job |
| `⌘K` / `Ctrl+K` | Open command palette |
| `⌘,` / `Ctrl+,` | Open settings |
| ``Ctrl+` `` | Toggle terminal drawer |

## Dashboard

| Shortcut | Action |
|----------|--------|
| `/` | Focus search / filter jobs |

## Job Creation & Transcript

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Submit prompt or operator message |

## Terminal

| Shortcut | Action |
|----------|--------|
| ``Ctrl+` `` | Toggle terminal (open/close) |

When the terminal is open and focused, pressing ``Ctrl+` `` closes it. When unfocused, it brings focus to the terminal.

## Command Palette

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate results |
| `Enter` | Select highlighted result |
| `Escape` | Close palette |
